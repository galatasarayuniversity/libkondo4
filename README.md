libkondo4
=========

A small C library to control the Kondo RCB-4 and ICS 3.0 protocols.

Requirements
------------

* libusb
* libftdi
* python-dev and swig for Python bindings

Building
--------

```
make
```

If you also want to build Python bindings, type

```
make -C python
```
